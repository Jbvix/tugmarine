1. Calculadora de Sondagem

Esta ferramenta realiza a interpolação volumétrica baseada nas tabelas de estaleiro (fichas de capacidade).

  • Entrada de Dados: Insira sempre a sondagem em CM (Centímetros). O sistema converte automaticamente para milímetros se o tanque exigir.
  • Correção de Trim: Introduza o Trim em MM (Milímetros). O motor utiliza Interpolação Bilinear para projetar o volume exato entre as faixas de trim da tabela.
  • Clamping: O sistema bloqueia valores acima da capacidade máxima (100% ou 95%) para evitar extrapolação matemática.

2. Assistente de Bunkering

Módulo logístico avançado para planeamento de abastecimento em lotes de 5000L.

  • Seleção de Tanques: Utilize os interruptores (toggles) para definir quais os tanques que participarão na operação.
  • Regra de 50 Litros: Cada tanque só aceita arredondamentos de 50L. Isto garante que a marcação na régua física seja clara.
  • Regra de 5000 Litros: O total consolidado é arredondado para o lote de caminhão mais próximo (Ex: Se houver espaço para 12.350L, o sistema autoriza 10.000L).
  • Régua Final Alvo: O valor em CM mostrado no card de resultado é a marca que deve ser atingida após o término do bombeio.

3. Checklist FOR-OPE-009

Conformidade total com as normas de segurança SAAM Towage.

  • Fluxo de Trabalho: Preencha as seções "Antes", "Durante" e "Após" sequencialmente.
  • Assinatura Digital: Utilize o ecrã tátil para assinar nos campos do Comandante e do Chefe de Máquinas.
  • Impressão: Utilize o botão "Gerar PDF" para criar uma cópia formatada A4 pronta para arquivo ou envio por email.
